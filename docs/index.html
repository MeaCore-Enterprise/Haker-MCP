¿Qué es Haker-MCP?

Es un puente entre los agentes de IA y tu sistema operativo. Los LLMs pueden ver, tocar y controlar tu computadora.

Agente IA (Claude, Cursor, etc.)
MCP Protocol
Haker-MCP Servidor
Sistema Mouse, Archivos, etc.
1
Agente IA envía una petición
2
Haker-MCP procesa vía MCP
3
Sistema ejecuta la acción

Capacidades

Todo lo que un agente IA puede hacer con Haker-MCP

Control de Mouse

Mover, click, arrastrar y scroll. Control total del cursor desde la IA.

Captura de Pantalla

Captura la pantalla completa o regiones específicas. Devuelve archivo o base64.

Sistema de Archivos

Leer, escribir y listar archivos. Manipulación completa del sistema de archivos.

Memoria Persistente

El agente puede memorizar, recordar y reflexionar sobre información entre sesiones.

Ejecución de Comandos

Ejecuta cualquier comando en la terminal del sistema y captura su salida.

Y mucho más

Notificaciones, clipboard, procesos, Docker, Wi-Fi, Excel, PDF, y 32 tools en total.

Instalación Rápida

3 pasos para tener tu MCP corriendo

1

Clona el repositorio

Descarga el código fuente desde GitHub

$ git clone https://github.com/KronoxYT/Haker-MCP.git
2

Instala dependencias

Instala las dependencias necesarias

$ cd Haker-MCP && npm install
3

Compila y ejecuta

Compila TypeScript y arranca el servidor MCP

$ npm run build && node dist/index.js
haker-mcp-terminal
$ npm run build && node dist/index.js
$ ▊

Configuración del Cliente MCP

Agrega Haker-MCP a tu cliente MCP favorito

En Cursor, ve a Settings → MCP Servers y agrega:

{
  "mcpServers": {
    "haker-mcp": {
      "command": "node",
      "args": ["C:\\ruta\\completa\\Haker-MCP\\dist\\index.js"]
    }
  }
}

En Claude Desktop, edita el archivo de configuración MCP:

{
  "mcpServers": {
    "haker-mcp": {
      "command": "node",
      "args": ["C:\\ruta\\completa\\Haker-MCP\\dist\\index.js"]
    }
  }
}

Archivo: %APPDATA%\Claude\claude_desktop_config.json

En Windsurf, agrega al archivo de configuración del proyecto:

{
  "mcpServers": {
    "haker-mcp": {
      "command": "node",
      "args": ["C:\\ruta\\completa\\Haker-MCP\\dist\\index.js"]

Seguridad

Usa Haker-MCP de forma responsable

Riesgo de Inyección

Los LLMs pueden ser engañados para ejecutar comandos maliciosos si los datos contienen instrucciones ocultas.

Entorno Aislado

Usa Haker-MCP solo en entornos de desarrollo con datos sintéticos, nunca en producción.

Supervisión Humana

La mayoría de clientes MCP requieren aprobación manual para cada tool call. Mantén esto activado.

Buenas prácticas

  • 01
    No conectes a producción — Usa solo en entornos de desarrollo con datos de prueba.
  • 02
    No compartas con usuarios finales — Es una herramienta de desarrollo interna.
  • 03
    Precaución con comandos — Revisa siempre qué va a ejecutar el agente antes de aprobar.
  • 04
    Protege la memoria — Los datos se guardan en ~/.haker_brain/. No almacenes información sensible.
